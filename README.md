## Cronômetro em JavaScript

Projeto simples de cronômetro desenvolvido em JavaScript puro, praticando lógica de programação, eventos e manipulação do DOM.

## Sobre o projeto

O cronômetro permite iniciar, pausar, continuar, resetar e marcar tempos durante a contagem. Também possui validações básicas para evitar marcações incorretas.

Este projeto foi desenvolvido com foco em aprendizado e pode ser utilizado como projeto de portfólio iniciante.

## Funcionalidades

Iniciar a contagem do tempo

Pausar e continuar o cronômetro

Resetar o tempo

Marcar tempos (laps)

Bloquear marcação quando o cronômetro estiver zerado

Desativar o botão de marcação automaticamente quando não pode ser usado

## Como funciona

O tempo é contado utilizando setInterval

A contagem é feita em centésimos de segundo (intervalo de 10ms)

O tempo é armazenado na variável timer

O valor é exibido no formato:

HH:MM:SS:CC

As marcações são armazenadas em um array e exibidas na tela
